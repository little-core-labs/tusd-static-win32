tusd-static-win32
=================

> static binaries for the `tusd` command on Windows.

## Installation

```sh
$ npm install tusd-static-win32
```

## Usage

```js
const tusd = require('tusd-static-win32')
console.log(tusd.path)
// '/home/werle/repos/little-core-labs/tusd-static-win32/bin/x64/tusd'
```
